## 📌 Project Overview

Smart Cyber Café Management & Billing Automation Platform is an **offline desktop-based management system** designed for cyber cafés and digital service centers.

The application digitizes day-to-day operations such as:

- 👤 Customer Registration
- 📁 Document Management
- 🧾 GST Billing & Invoice Generation
- 📊 Excel Report Generation
- 🔍 Instant Document Retrieval
- 📜 Activity Logging
- 🔐 Secure Authentication

The system eliminates paper-based operations and provides a fast, reliable, and professional workflow.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/smart-cyber-cafe.git
```

```
cd smart-cyber-cafe
```

---

## Create Virtual Environment

```
python -m venv venv
```

Windows

```
venv\Scripts\activate
```

Linux

```
source venv/bin/activate
```

---

## Install Dependencies

```
pip install -r requirements.txt
```

---

## Run Project

```
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
